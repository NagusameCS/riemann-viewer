Text → Tensor & Manifold — Guide

What is happening?

Your text is passed through a deterministic embedding pipeline — the same kind of computation that happens inside every large language model. Each token (word or character) is mapped to a 24-dimensional vector, mixed with its neighbors via a simplified attention mechanism, and then projected to 3D via PCA to visualize as a Riemannian manifold.

Embedding Tensor (left panel)

A grid where each row = one token and each column = one embedding dimension (0–23). Color = value: blue = negative, yellow = near zero, red = positive. This is exactly what a real model's hidden state looks like after an embedding layer + layer norm.

Similarity Matrix (Similarity tab)

The cosine similarity between every pair of token embeddings — how similar two tokens are in the embedding space. Diagonal = 1.0 (self-similarity). Bright off-diagonal entries mean the model "sees" those words as related.

Riemannian Manifold (right panel)

The 3D projection of the token embeddings forms a point cloud. A Delaunay triangulation builds a surface through these points. The scalar curvature R is computed as the discrete Laplacian of the embedding trajectory:

  • Blue — R < 0: saddle-like (hyperbolic), token is bending toward the center
  • Yellow — R ≈ 0: flat, smooth transition
  • Red — R > 0: sphere-like (elliptic), token is on an outer curvature peak

The glowing tube traces the sequence path — where the "token" travels through embedding space from first to last word.

What is actually computed?

  1. Base embedding: Each character's unicode code point is hashed deterministically into a 24-dim vector. Words are averaged over their characters.
  2. Positional encoding: Sinusoidal position signals (like real transformers use) are added.
  3. Layer norm: Embeddings are normalized to zero mean, unit variance.
  4. Context mixing: Each token attends to all others via a simplified dot-product attention (σ-weighted sum), capturing semantic neighborhood.
  5. PCA: The 24-dim space is projected to 3D via power-iteration PCA. The variance explained shows how much geometric structure survives the projection.
  6. Curvature: Ri = ‖ei+1 − 2ei + ei−1‖ (discrete Laplacian). Sign: positive if the token lies on the outer shell, negative if near the centroid.
